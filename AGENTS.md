# Agent Instructions

This repository contains a small static HTML/JavaScript canvas game.

- Do not rewrite the whole game unless explicitly requested.
- Do not redesign the game or change its visual style.
- Preserve the current art direction and existing image assets.
- Keep comments and logs in English.
- Keep changes small, reviewable, and testable.
- Fix deployment issues and obvious broken references only.
- Do not change gameplay unless required to fix clearly broken behavior.
- Keep the project deployable as static hosting only unless the user explicitly requests a backend or build system.
- Prefer relative paths for scripts and assets.
- Verify the game from a local static server after changing runtime files.
